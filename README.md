## What This Does

An automated content calendar that scrapes, curates, and displays daily AI and tech news. GitHub Actions runs on a schedule to fetch fresh content, and GitHub Pages serves a clean dashboard for browsing topics and planning content.

---

## How It Works

```
GitHub Actions (scheduled cron)
        │
        ▼
Python scripts fetch AI/tech news
        │
        ▼
Data saved to /data directory (JSON)
        │
        ▼
index.html renders dashboard
        │
        ▼
GitHub Pages serves live site
```
